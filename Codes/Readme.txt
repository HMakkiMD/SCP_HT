1. The "ML" folder provides the input data (cvs file) and code for the Machine Learning study.
2. The "DOS_LL" folder provides the code (band_tail.py) to calculate bandwidth, tail-width, and localisation length at the band edge. 
3. The codes for calculating the persistence length and the planarity persistence length (using various methods explained in the paper) are provided in the "Lp_Lpp" folder.
4. The code for calculating the dipole moment of the repeat unit is given in the "Dipole" folder.
